ZZT Crashopedia!

This archive contains the following ZZT worlds:
	- ZZTCRASH.ZZT: The main Crashopedia file. Play it to enter a menu
	  showing different ways to crash and hang ZZT.

	- MONIBOMB.ZZT: Monitor Bomb, a way to make a title screen hang.

	- NEGABORD.ZZT: A world with a negative board count. Crashes ZZT
	  immediately on loading.

It also contains a LICENSE (CC-BY-SA 4.0) and this README file.
